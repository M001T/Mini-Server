#  🎒 Mini-Server
Um Simples CRUD/Server feito com Ruby

<h3> About 📃</h3>

Simples e para uso de Estudo!

<h3> Kernel 📦</h3>

- Ruby 2.6.3+


<h3> Install 💻</h3>

Setup
set configuration, db ,etc

config/config.rb

install gems

rake gem:install

create table db

rake db:migrate

Test
rspec

rake run:rspec

Run
Terminal : rake run:term

Web : rake run:web
