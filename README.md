#  :pencil2: Mini-Server
Um Simples CRUD/Server feito com Ruby

<h3> Sobre 📃</h3>

Simples e para uso de Estudo!

<h3> Kernel 📦</h3>

- Ruby 2.6.3+


<h3> Install 💻</h3>

## Precisa de:
1. Ruby
2. Postgresql (qualquer SQL DB)

## Setup 🔧
setar config de db etc...
> ```config/config.rb```

Instalar Gems 💎
> ```rake gem:install```

Criar Tabela db 📊
> ```rake db:migrate```

## Test 🔨
rspec
> ```rake run:rspec```

## Run 💻
Terminal :
```rake run:term```

Web :
```rake run:web```
